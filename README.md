# Proyecto-An-lisis-de-Ingresos-y-Gastos

Descripción
Proyecto de Excel enfocado en el análisis de ingresos, gastos y clientes, mediante el uso de tablas dinámicas, automatización de consultas con BUSCARX y un dashboard interactivo para la toma de decisiones financieras.

Puntos clave del proyecto

Gestión de datos financieros
Base de datos con 5,000 transacciones financieras simulando un entorno empresarial real
Registro de ingresos, gastos, clientes, impuestos, bancos y métodos de pago

Análisis de ingresos y gastos
Comparación de ingresos vs gastos por período
Identificación de categorías con mayor impacto financiero

Análisis de clientes
Identificación de clientes más rentables (Top 10)

Automatización con BUSCARX
Consulta automática de transacciones mediante ID
Recuperación instantánea de información clave (cliente, monto, banco, fecha y estado)
Reducción de búsqueda manual y errores

Dashboard financiero interactivo
Gráficos de Gastos vs Ingreso, Gastos por categoría y Top 10 Clientes
